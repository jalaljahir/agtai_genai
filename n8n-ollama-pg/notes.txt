# connect postgres to n8n - 
find the host name by running the /conninfo command OR the following qry in pgadmin:
SELECT inet_server_addr();




pgAdmin: Access pgAdmin by going to http://localhost:5050 in your browser. Log in with the email and password you configured in your docker-compose.yml file.
To add a connection, click Add New Server, give it a name, and go to the Connection tab. Use the following details:
Host name/address: postgres-pgvector
Port: 5432
Username: n8nuser
Password: n8npass
Click Save. 

# check pg vector extension
SELECT * FROM pg_extension WHERE extname = 'vector';
# if no results then run the following to create extension
CREATE EXTENSION IF NOT EXISTS vector;
